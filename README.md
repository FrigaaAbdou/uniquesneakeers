# UniqueSneakeers

UniqueSneakeers is a premium sneaker storefront landing page built to showcase rare and authentic footwear in a modern, high-impact way. The experience is designed around a dark visual language, a swipeable 3D hero carousel, and product-focused presentation that feels editorial rather than generic.

## Overview

This project presents a mobile-first sneaker website concept with:

- A cinematic intro transition
- A moving announcement marquee
- A 3D-style swipeable hero carousel using transparent product images
- A centered "Just Landed" product section
- Premium dark-mode styling and motion-led interactions

The goal of the project is to create a storefront that feels polished, current, and visually strong while keeping the interface clear and conversion-focused.

## Tech Stack

- HTML
- Tailwind CSS via CDN
- Vanilla JavaScript
- Local product assets

## Running Locally

Open the project with any local static server. For example:

```bash
python3 -m http.server 4173
```

Then visit `http://127.0.0.1:4173/`.

## Features

- Dark premium UI tailored for sneaker culture
- Production-style swipe interactions in the hero carousel
- Layered 3D carousel behavior with keyboard, pointer, and autoplay handling
- Transparent sneaker cutouts with white glow treatment
- Responsive layout built for mobile first, then scaled for larger screens
